# Face Recognition System

A web-based face recognition system that identifies and tracks individuals using real-time camera feed. The system stores facial data in a database and provides instant recognition when a person appears on camera.

## Features

- **Face Registration**: Add new individuals to the database with their photos
- **Real-time Recognition**: Automatically identify registered persons through webcam/camera feed
- **Web Interface**: Easy-to-use web application for management and monitoring
- **Database Storage**: Persistent storage of facial encodings and user information


## Usage

### Adding a New Person

1. Navigate to the "Add Person" page
2. Enter person's name and description
3. Upload their photo
4. Submit to save to database

### Recognition

1. Go to the Home page
2. Allow camera access when prompted
3. The system will automatically detect and identify registered faces
4. Recognition results appear in real-time on screen


## Known Limitations

- Recognition accuracy depends on lighting conditions
- Requires clear frontal face images for best results
- Database grows with number of registered users
